# Cocoa

## 简介

[AeCw](https://github.com/AECBanana) 的第三代 osu! 皮肤，专为 AeCw 定制。本仓库只是代为存档。

> UI 是随便造的，以后会完善，主要是游玩界面，随着实力提高做出改变。目前稳定版本为这个白色 note 的类 DT 皮，光标采用 RGB(255,0,255) 洋红色，更醒目。彩色版本的 note 还在摸索。
>
> 皮肤为 JUMP 特化，祝你大跳全连上~

## 素材来源
